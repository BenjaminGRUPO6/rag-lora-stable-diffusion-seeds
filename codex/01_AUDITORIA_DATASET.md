# Tarea para Codex

## Objetivo

Implementa y valida la auditoría del dataset.

## Criterios de aceptación

- Completar `src/data/audit.py` y `scripts/audit_dataset.py`.
- Generar CSV/JSON de archivos, clases, resoluciones, corruptos y duplicados.
- No eliminar ni mover imágenes.
- Añadir pruebas y documentación.

## Entrega esperada

- Lista de archivos modificados.
- Comandos de prueba ejecutados.
- Riesgos o pendientes reales.
- Diff revisable antes de commit.
