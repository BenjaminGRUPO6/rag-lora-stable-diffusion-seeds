# Uso de Codex local

Ejecute Codex desde la raíz del repositorio. Trabaje una tarea por rama y revise el diff antes de aceptar cambios.

Orden recomendado:

1. `00_PROMPT_MAESTRO.md`
2. `01_AUDITORIA_DATASET.md`
3. `02_ETIQUETADO_SPLIT.md`
4. `03_ENTRENAMIENTO_VISUAL.md`
5. `04_PREPARACION_LORA.md`
6. `05_ENTRENAMIENTO_LORA_SD15.md`
7. `06_CONSTRUCCION_RAG.md`
8. `07_INTEGRACION_APP.md`
9. `08_EVALUACION_DOCUMENTACION.md`

No pida a Codex que ejecute todas las fases en un solo cambio. Cada prompt tiene criterios de aceptación.
