# Métricas

## Modelo visual

- Accuracy.
- Macro-F1.
- Precision y recall por clase.
- Matriz de confusión.
- F1 de clases minoritarias.

## LoRA y datos sintéticos

- Pérdida de entrenamiento.
- Evaluación humana de calidad visual (1-5).
- Fidelidad a la clase/prompt (1-5).
- Tasa de aceptación de imágenes sintéticas.
- Comparación del clasificador antes/después de usar sintéticos.

## RAG

- Precision@k.
- Recall@k.
- MRR.
- Porcentaje de afirmaciones respaldadas por fuentes.
- Latencia.

## Sistema

- Tiempo total por consulta.
- Evaluación humana del informe.
- Tasa de respuestas que incluyen advertencia y fuentes.
