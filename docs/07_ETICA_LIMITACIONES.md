# Ética y limitaciones

- Una fotografía no confirma una enfermedad ni reemplaza análisis de laboratorio.
- El sistema debe mostrar nivel de confianza y casos en que no debe usarse.
- Los documentos del RAG deben tener procedencia, fecha y alcance.
- El dataset puede contener sesgos por especie, iluminación o equipo fotográfico.
- Las imágenes sintéticas deben estar claramente marcadas y no presentarse como evidencia real.
- Se necesita supervisión humana para aceptar sintéticos y revisar informes.
- Deben respetarse licencias, propiedad intelectual y privacidad de los datos.
