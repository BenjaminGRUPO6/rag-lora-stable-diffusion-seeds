# Ética y limitaciones

- La clasificación es visual y preliminar.
- La categoría `spotted` no identifica automáticamente un patógeno.
- Toda recomendación debe mostrar fuente.
- Debe existir supervisión humana.
- Las imágenes sintéticas deben identificarse como tales.
- No se deben usar sintéticos en validación o prueba.
- Se debe respetar la licencia y citar el dataset.
- El sistema debe abstenerse cuando no haya evidencia suficiente.
- Se documentarán sesgos, condiciones de captura y clases no cubiertas.
